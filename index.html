Mixlet

A mini player for YouTube Music on macOS.

A focused, lightweight way to keep music close without living in a browser tab.

via the Mac App Store

Why Mixlet?

How is Mixlet better than using Safari's Add to Dock portable web app feature with YouTube Music?
  • MINI: Mini player first, not an afterthought.
  • EFFICIENT: Full set of keyboard shortcuts and media key support.
  • RESUME: Resume playback at the same song and timestamp.
  • MAC-LIKE: Clean, focused mac-assed UI.
